# ColonAI - DocMD.io Setup Guide

Quick guide untuk import dokumentasi ke DocMD.io.

## 📂 File Structure

Semua dokumentasi ada di:
```
docs-ui/docs/
├── intro.md
├── getting-started.md
├── architecture.md
├── agents.md
├── scout-agent.md
├── momentum-agent.md
└── ... (semua file markdown lain)
```

## 🚀 Cara Setup DocMD.io

### Step 1: Connect GitHub
1. Buka [docmd.io](https://docmd.io)
2. Sign up/login dengan GitHub
3. Connect repository: `labsmula/ColonAI`

### Step 2: Import Files
1. Pilih folder: `docs-ui/docs/`
2. Import semua file markdown
3. Tunggu processing (otomatis)

### Step 3: Configure
1. Set project name: ColonAI
2. Set custom domain: `colonai.docmd.io` (optional)
3. Set theme: GitBook (default sudah pas)

### Step 4: Done!
- Docs langsung online di `https://colonai.docmd.io`
- Auto-update dari GitHub setiap push

## ✅ Benefits

- ✅ Zero setup time (langsung import)
- ✅ No build errors (platform handle semua)
- ✅ GitBook-like UI
- ✅ Built-in search
- ✅ Auto-sync dengan GitHub
- ✅ Collaborative editing
